# Rock Paper Scissors Game
Rock Paper Scissors Game completed on JavaScript. Allows user to compete in a game of RPS with computer, keeping score of each result. Project runnable with Live Server extension.

## Technologies Used:

### JavaScript

JavaScript was used to write the logic of the game. The playGame() function takes player choice (user input) from HTML buttons and compares it against random results generated by the Math class. If and switch statements written to iterate through all options, with ternary operators used to compare user input and display output for all cases. Also used to display scores of all results.

### HTML

Used to create the titles, score displays with span components, and buttons to listen to user choice with mouse events.

### CSS

Used for styling of background, buttons, score displays, hover and active changes, and overall layout.
